# ROS2 Learning Plan

## Objective

Build enough ROS2 foundation to safely explore GO2 command workflows and future AI-agent integration.

## Learning Topics

- ROS2 workspace structure
- Nodes
- Topics
- Services
- Actions
- Launch files
- Parameters
- Message types
- Logging
- Simulation-first testing where possible

## Practice Sequence

1. Install ROS2 on Ubuntu.
2. Run official beginner examples.
3. Create a simple publisher and subscriber.
4. Create a service example.
5. Create an action example.
6. Document the command schema needed before touching hardware.
7. Move to GO2 connectivity only after local basics are understood.

## Public Evidence To Add Later

- Setup screenshot.
- Simple ROS2 node example.
- Notes from a successful local ROS2 run.
- GO2-safe command schema draft.
- Short demo video after safe testing.

## Claim Boundary

This learning plan does not claim completed ROS2 integration with GO2.
